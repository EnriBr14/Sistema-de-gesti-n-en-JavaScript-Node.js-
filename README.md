# Sistema de gestión - Node.js

## Descripción
Aplicación de consola desarrollada en Node.js que permite gestionar entidades mediante programación orientada a objetos, validaciones y lógica de negocio.

## Tecnologías
- Node.js
- JavaScript (ES6)
- Programación Orientada a Objetos

## Funcionalidades
- Registro de entidades
- Validación de duplicados
- Listado de datos
- Eliminación de registros
